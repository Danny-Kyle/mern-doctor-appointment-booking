doctor Model

userID, 
first and last name, 
work email, 
phone number, 
official website, 
address, 
field of specialisation, 
experience
 and status

charge per meeting
which will be set to type Number,
and is required

convenient meeting time
which will be an array of objects so set to type array
also required
